# 🍿 Fast Finder - Advanced Auto Filter Bot & Web Panel

Fast Finder एक अल्ट्रा-एडवांस्ड, हाई-परफॉर्मेंस टेलीग्राम ऑटो फ़िल्टर बॉट है, जो एक बेहद खूबसूरत **टेलीग्राम मिनी ऐप (Mini App)** और **वेब डैशबोर्ड (Web Dashboard)** के साथ आता है। यह प्रोजेक्ट पूर्णतः **Hydrogram Async** पर आधारित है और इसमें कोएब (Koyeb Free Tier) के लिए विशेष **रैम और डिस्क ऑप्टिमाइज़ेशन** किया गया है।

---

## 🚀 मुख्य फीचर्स (Core Features)

### 🤖 टेलीग्राम बॉट फीचर्स
* **Smart Auto Filter:** ग्रुप में मूवी या सीरीज़ का नाम लिखते ही सटीक बटन रिजल्ट्स (12 परिणाम प्रति पेज)।
* **Gemini 2.5 Flash AI:** 10 मिनट की चैट हिस्ट्री मेमोरी के साथ एडवांस्ड विज़न (इमेज एनालिसिस) सपोर्टेड एआई असिस्टेंट (`/ai`, `/ask`)।
* **Anti-Abuse Rate Limiter:** कमांड स्पैमिंग को रोकने और कोएब CPU को 100% ब्लास्ट होने से बचाने के लिए सख्त कूलडाउन मैकेनिज्म।
* **Persistent Auto-Delete Engine:** इन-मेमोरी टाइमर के बजाय रीस्टार्ट-प्रूफ मोंगोडीबी कतार (MongoDB Queue)। कोएब रीस्टार्ट होने पर भी ग्रुप की लिंक्स और मैसेजेस समय पर डिलीट होंगे।
* **YouTube Studio Flow (Edit/Crop):** एडमिन पैनल के अंदर लाइव पिंच-ज़ूम क्रॉपर की मदद से पोस्टर्स को क्रॉप करके सीधे टेलीग्राम थंबनेल को लाइव ओवरराइड करने की क्षमता।

### 📱 मिनी ऐप और वेब डैशबोर्ड
* **Telegram Native UI Theme:** यूज़र के टेलीग्राम ऐप की थीम (लाइट/डार्क/कस्टम) के साथ मिनी ऐप का ऑटो-सिंक लेआउट।
* **Native Progress Bars:** पेज चेंज या लिंक जनरेट करते समय टेलीग्राम के नेटिव `MainButton` लोडिंग बार का उपयोग।
* **Parallel Thumbnails Engine:** थंबनेल डाउनलोडिंग को तेज़ करने के लिए 5-समानार्थी सेमाफोर कंट्रोल।
* **Streaming & Download Architecture:** वीडियो प्लेयर (Plyr.js) में 10-सेकंड यूट्यूब स्टाइल डबल-टैप जेस्चर और IDM सपोर्टेड फ़ास्ट डाउनलोड पाइपलाइन।

## 🛠️ एनवायरनमेंट वेरिएबल्स (Environment Variables)

Koyeb पर डिप्लॉय करते समय निम्नलिखित वेरिएबल सेट करना अनिवार्य है:

| कुंजी (Key) | विवरण (Description) | डिफ़ॉल्ट/उदाहरण (Example) |
| :--- | :--- | :--- |
| `API_ID` | टेलीग्राम एपीआई आईडी | `1234567` |
| `API_HASH` | टेलीग्राम एपीआई हैश | `abcdef123456...` |
| `BOT_TOKEN` | टेलीग्राम बॉट फ़ादर टोकन | `12345:AAH_...` |
| `ADMINS` | बॉट मालिकों/एडमिन्स की आईडी (स्पेस से अलग) | `12345678 98765432` |
| `DATABASE_URL` | मोंगोडीबी एटलस कनेक्शन स्ट्रिंग | `mongodb+srv://...` |
| `DATABASE_NAME` | मोंगोडीबी डेटाबेस का नाम | `Cluster0` |
| `LOG_CHANNEL` | बॉट के स्टार्टअप और यूजर लॉग्स का चैनल | `-100xxxxxxxxx` |
| `BIN_CHANNEL` | स्ट्रीमिंग और थंबनेल जनरेशन का गुप्त चैनल | `-100xxxxxxxxx` |
| `URL` | कोएब ऐप का डोमेन यूआरएल (HTTPS के साथ) | `https://app-name.koyeb.app/` |
| `GEMINI_API_KEY` | गूगल जेमिनी एआई की एपीआई की | `AIzaSy...` |
| `UPI_ID` | प्रीमियम एक्टिवेशन के लिए यूपीआई आईडी | `yourname@upi` |
| `UPI_NAME` | यूपीआई पर आपका मर्चेंट/ओरिजिनल नाम | `John Doe` |
| `RECEIPT_SEND_USERNAME` | पेमेंट स्क्रीनशॉट प्राप्त करने वाले एडमिन का यूज़रनेम | `@AdminUsername` |
